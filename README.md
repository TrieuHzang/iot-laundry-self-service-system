# iot-laundry-self-service-system
A smart, end-to-end IoT self-service laundry prototype featuring a dual-MCU architecture (STM32 &amp; ESP32). It automates unmanned laundry operations by integrating an online VietQR payment gateway (via payOS), a responsive React web app, and Blynk IoT cloud communication to securely monitor and control physical washing/drying components in real-time.
